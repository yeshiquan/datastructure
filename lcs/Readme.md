LCS问题：

给定序列

X = <x1,x2,...,xn>

和另一个序列

Y = <y1,y2,...,ym>

 

找两个递增的下标序列

<i1, i2, ...ik> 和 <j1, j2, ..., jk>使

xi1 == yj1

xi2 == yj2

......

xik == yjk

令Z = <xi1, xi2,..., xik>，那么称Z是X和Y的一个公共子串，

LCS问题是求最长的公共子串，即最长的下标序列

 

动态规划解法：

令Xi表示X的前缀<x1,x2,...,xi>

c[]i[j] 表示Xi和Yi的LCS长度，动态规划的状态转移方程为：

如果xi == yj， c[i][j] = c[i-1][j-1] + 1 

如果xi != yj,      c[i][j] = max(c[i-1][j], c[i][j-1])
